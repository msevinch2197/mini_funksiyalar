# Parametrli funksiyalar 

Ushbu loyiha C++ dasturlash tilida yozilgan bo‘lib, parametrli funksiyalarni o‘rganish uchun mo‘ljallangan.

## 📌 Vazifalar

### 1. Ism bilan salom

Foydalanuvchidan ism qabul qilib, quyidagi xabarni chiqaradi:

```
Salom, [ism]!
```

### 2. Raqam chiqarish

Berilgan sonni konsolga chiqaradi.

### 3. Ikkita raqam qo‘shish

Ikki son qabul qilib, ularning yig‘indisini chiqaradi.

### 4. Ism va yosh

Ism va yosh qabul qilib, quyidagi ko‘rinishda chiqaradi:

```
[ism] [yosh] yoshda.
```

---

## ▶️ Ishga tushirish

1. Faylni kompilyatsiya qiling:

```
g++ main.cpp -o app
```

2. Dasturni ishga tushiring:

```
./app
```

